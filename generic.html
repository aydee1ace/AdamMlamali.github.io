Motor Insurance Claims Project

Classification & Regression Analysis of Car Insurance Claims

Introduction

This project uses a dataset sourced from: Kaggle - Car Insurance Claim Data. The dataset contains approximately 8,000 records of car insurance policyholders. It includes features such as:

  • Age
  • Income
  • Gender
  • Claim history
  • And more

The dataset includes two target variables:

  1. Whether a claim has occurred (binary classification)
  2. The total cost of the claim, if one has occurred (regression)

Objective

The aim of this project is to solve two core problems:

  1. Classification: Predicting if a policyholder will make a claim.
  2. Regression: Estimating the total value of a claim, assuming one occurs.

Performance

Regression model performance:

  • RMSE: 8,383.78
  • MAE: 3,515.91
These results indicate that while the RMSE on the test set was slightly higher than on the training data, it is still fairly consistent. There is room for further improvement through:
  • Additional feature engineering
  • Investigating feature importance
  • Creating ensemble models
